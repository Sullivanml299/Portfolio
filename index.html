OneScreen

Interactive 3D tools for teaching dental anatomy.

Technical Lead Web · iOS · Android

An instructor-facing web app in TypeScript with React Three Fiber and Vite, paired with student iOS and Android clients in Unity — all on a Supabase backend. It combines an interactive 3D model lecturing tool with a class-management system where instructors create, assign, and review tooth- and feature-identification assignments. I lead the project technically and build across the full stack.

Runtime Image Classification

An on-device CNN that reads hand-drawn runes.

ML & Systems Unity · Sentis

A runtime image-classification system in Unity using Sentis to run a CNN that recognizes player-drawn runes. I used PyTorch and transfer learning to retrain an open-source model on hand-drawn runes, plus standard Python tooling to augment the training set for robust recognition.

Journal System

A writable, page-turning 3D journal.

Graphics & Tools Unity · Shaders

A Unity system that lets a player in a 3D world write and draw on the 2D pages of a journal. I built the pages from Bézier patches and animated page turns by manipulating vertex positions directly, and wrote a custom color picker as a 3D cube with a shader that maps model space to color space.

Unity· C#· Shaders· Bézier Surfaces
